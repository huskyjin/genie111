# Project-Scoped Rules

1. **명시적 지시 범위 외 임의 수정 금지**: 사용자가 지시한 요구사항 외에 이전에 구현 완료된 다른 레이아웃이나 로직을 임의로 롤백, 제거, 리팩토링하지 않습니다. 명시적으로 지시하지 않은 코드는 보존합니다.
2. **국소적 편집 지향**: 변경이 필요한 부분만 정확히 짚어 안전하게 편집하며 전체 롤백 등 파괴적 행위를 차단합니다.
3. **프로토타입 경량화 상시 유지**: 프로토타입의 핵심 목적인 유저 플로우 검증을 원활하게 진행할 수 있도록, 복잡한 모의 데이터 연산 및 파일 처리 코드를 제거하고 순수한 UI 껍데기 인터랙션 중심의 경량 상태를 상시 유지합니다.
